## Before/After Transformations

### Example 1: Occupancy Trend

**❌ BEFORE** (Weak):
```
The company has strong occupancy trends showing positive momentum
```

**✅ AFTER** (High-Impact):
```
Occupancy jumped 280bps to 96.8% (highest in 5 years), driven by supply-constrained
sunbelt markets with 24-month lease duration runway providing SS-NOI visibility through 2026
```

**Why it works**:
- Specific metric: 280bps gain, 96.8% level
- Context: "highest in 5 years"
- Driver: "supply-constrained sunbelt markets"
- Forward-looking: "24-month runway through 2026"

---

### Example 2: Dividend Yield

**❌ BEFORE** (Weak):
```
The REIT offers an attractive dividend yield with good coverage
```

**✅ AFTER** (High-Impact):
```
5.2% dividend yield vs sector median 4.1% (120bps premium), covered at 75% AFFO payout
(best-in-class), implying 200bps annual distribution growth potential without leverage increase
```

**Why it works**:
- Absolute metric: 5.2% yield
- Comparison: vs sector median 4.1%
- Coverage quality: 75% payout vs implied 80% threshold
- Implication: 200bps growth runway

---

### Example 3: Development Pipeline

**❌ BEFORE** (Weak):
```
The company has a significant development pipeline that should drive growth
```

**✅ AFTER** (High-Impact):
```
$1.2B development pipeline delivering in Q3 2025 at 8.5% stabilized yield vs 6.2%
portfolio cap rate = $24M annual NOI accretion (4.2% AFFO/share boost)
```

**Why it works**:
- Size: $1.2B quantified
- Timing: Q3 2025 delivery date
- Economics: 8.5% yield vs 6.2% portfolio = 230bps spread
- Impact: $24M NOI = 4.2% AFFO accretion

---

### Example 4: Debt Maturity Risk

**❌ BEFORE** (Weak):
```
The company faces some refinancing risk in coming years
```

**✅ AFTER** (High-Impact):
```
$2.1B debt maturity wall in 2026 (18% of EV) faces 200bps higher refinancing costs
vs legacy 3.5% coupons, implying $42M annual AFFO headwind (5% drag on growth)
```

**Why it works**:
- Magnitude: $2.1B, 18% of EV
- Timing: 2026 specific
- Cost: 200bps rate increase quantified
- Impact: $42M AFFO drag = 5% growth headwind

## Metric Formatting Standards

### Price Targets
```
❌ "Price target around 50 dollars"
✅ "$52.50" (mono font, gold color, large size)
```

### Percentages
```
❌ "Five point two percent"
✅ "5.2%" (mono font, context required)
```

### Growth Rates
```
❌ "Strong growth"
✅ "8.2% AFFO/share CAGR" (with time period)
```

### Dollar Amounts
```
Small: "$24M" (millions)
Medium: "$1.2B" (billions)
Large: "$45.8B" (billions)
Precise: "$1,247M" (for specific calculations)
```

### Basis Points
```
❌ "2%"
✅ "200bps" (when discussing changes/spreads)
```

### Ratios/Multiples
```
❌ "Four times"
✅ "4.2x" (mono font, with context like "Net Debt/EBITDA")
```

### Cap Rates
```
❌ "Six and a half percent cap rate"
✅ "6.5% cap rate" (always include "cap rate" label)
```

### Occupancy
```
❌ "Ninety-six percent occupied"
✅ "96.2% occupancy" (to decimal point for precision)
```

### Payout Ratios
```
❌ "Seventy-five percent payout"
✅ "75% AFFO payout" (always specify AFFO vs FFO)
```

## Common Pitfalls to Avoid

### 1. Vague Adjectives
```
❌ "Strong performance"
❌ "Robust growth"
❌ "Attractive valuation"
❌ "Solid fundamentals"
❌ "Impressive results"

✅ "8.2% AFFO growth vs 4.5% sector median"
✅ "Trading at 0.85x NAV vs 0.95x peer average"
✅ "96.8% occupancy, up 280bps in 12 months"
```

### 2. Missing Context
```
❌ "5.2% dividend yield"
✅ "5.2% dividend yield vs 4.1% sector median"

❌ "High occupancy of 96%"
✅ "96.2% occupancy vs 10-year avg of 93.5%"

❌ "Significant development pipeline"
✅ "$1.2B pipeline (12% of GAV) vs 8% sector norm"
```

### 3. No Time Horizon
```
❌ "Development will drive growth"
✅ "Development delivering Q3 2025 drives 4.2% AFFO boost"

❌ "Debt maturities are manageable"
✅ "Only 8% of debt maturing before 2027"

❌ "Rent growth expected to improve"
✅ "Lease spreads projected to reach 5% by Q2 2025"
```

### 4. Unquantified Risks
```
❌ "Rising rates pose a risk"
✅ "$2.1B debt (18% of EV) refinancing at +200bps = $42M AFFO drag"

❌ "Occupancy could decline"
✅ "850K SF expiration risk could trigger 480bps occupancy drop = $68M NOI loss"

❌ "Supply headwinds exist"
✅ "450M SF supply (8% of stock) risks 150-200bps cap rate expansion"
```

### 5. Generic Bull/Bear Cases
```
❌ Bull: "Good management team"
❌ Bull: "Strong balance sheet"
❌ Bear: "Macro uncertainty"
❌ Bear: "Competitive pressure"

✅ Bull: "Net Debt/EBITDA of 4.2x (best-in-class) enables $3B acquisition capacity at 50bps WACC spread"
✅ Bear: "$8.5B debt maturity in 2026-27 (22% of EV) faces 175bps refinancing headwind = $149M AFFO drag"
```

### 6. Missing Implications
```
❌ "SS-NOI growth of 6.5%"
✅ "6.5% SS-NOI growth vs 3.2% sector avg provides 18-month visibility through 2026, supporting 8%+ AFFO CAGR"

❌ "75% AFFO payout ratio"
✅ "75% AFFO payout vs 80% threshold = 200bps distribution growth potential without leverage increase"

❌ "Development at 8.5% yield"
✅ "8.5% development yield vs 6.2% portfolio cap = 230bps spread = $24M annual NOI accretion"
```

### 7. Inconsistent Units
```
❌ Mixing "$24 million" and "$1.2B"
✅ Consistent: "$24M" and "$1.2B"

❌ Mixing "2%" and "200 basis points"
✅ Use bps for changes: "200bps increase to 5.5%"

❌ Mixing "four times" and "4.2x"
✅ Always use "x" notation: "4.0x" and "4.2x"
```

### 8. Buried Lead
```
❌ "The company has been working on improving occupancy and has seen some positive results recently with rates moving higher"
✅ "Occupancy jumped 280bps to 96.8% in 12 months (highest since 2019)"

❌ "Development projects are progressing and are expected to deliver significant value creation over time"
✅ "$1.2B pipeline delivering Q3 2025 at 8.5% yield = $24M NOI accretion (4.2% AFFO boost)"
```

## Quick Reference: Insight Quality Checklist

Before publishing any AI-generated insight, verify:

- [ ] **Quantified**: Does it include specific numbers?
- [ ] **Contextualized**: Is it compared to something (peer, history, sector)?
- [ ] **Time-Bound**: Does it specify when (dates, periods)?
- [ ] **Causal**: Does it explain the driver/reason?
- [ ] **Actionable**: Does it inform a decision?
- [ ] **Sourced**: Is there a verifiable URL?
- [ ] **Material**: Does it matter (>5% impact)?
- [ ] **Concise**: Is it under 50 words?
- [ ] **Clinical**: Zero marketing language?
- [ ] **Formatted**: Proper units ($M, %, bps, x)?

**Scoring**:
- 10/10 = Excellent, publish immediately
- 7-9/10 = Good, minor edits
- 4-6/10 = Needs revision
- <4/10 = Regenerate with better prompt
